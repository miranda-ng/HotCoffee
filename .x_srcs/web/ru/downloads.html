Самостоятельная компиляция последней версии

Многие люди во всех уголках нашей планеты всегда наслаждались, наслаждаются и будут наслаждаться вкусом и ароматом кофе.
И так сложились обстоятельства, что наша сборка получила имя в честь этого замечательного напитка, и также к имени получила прилагательное, которое намекает на готовность к употреблению этого напитка.
В связи с вышесказанным, всем желающим предоставлена возможность самостоятельно "варить кофе" из "свежемолотых зёрен арабики".
Для этого требуется клонировать проект сборки HotCoffee и запустить файл "build.cmd". После завершения будет скомпилировано два установочных файла (x64 и x86) в папке "output".

Обновление без установщика

Обновить все компоненты Miranda NG можно с помощью плагина PluginUpdater:
Главное меню → Обновить → Проверить обновления
Сразу после обновления рекомендуется установить любимый набор значков:
Главное меню → Вид → Значки → набор значков
