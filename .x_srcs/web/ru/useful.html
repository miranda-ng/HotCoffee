Полезно знать

Для того чтобы в контакт-листе отображались контакты какого-то определённого протокола, нужно с зажатой клавишей Ctrl щёлкнуть ЛКМ на значке нужного протокола, в панели статуса контакт-листа.
Снять этот режим просмотра можно щелчком ЛКМ в свободном месте панели статуса (между или рядом со значками протоколов).
Если вы начали набирать текст в неправильной раскладке:
  • Раскладка набранного текста меняется комбинацией клавиш Ctrl + R.
  • При нажатии клавиш Ctrl + Y вы измените регистр набранного текста.
  • Также при нажатии клавиш Ctrl + T текст будет написан транслитом.
  • Если вместо клавиши Ctrl зажимать клавишу Alt, то все операции будут совершаться только с последним словом.
Примечание: требуется плагин Раскладка клавиатуры.
Вы можете быстро отправить скриншот экрана своему собеседнику. Для этого нажимаем клавишу Print Screen (или PrtScr ⎙), ставим курсор в поле ввода текста и нажимаем Ctrl + V. В результате появится окно передачи файла (вашего скриншота) собеседнику.
При наведении мыши на контакт появляется всплывающая подсказка. Из неё можно легко скопировать нужные данные, нажав Ctrl + C и выбрав нужный пункт.
Примечание: требуется плагин Подсказки.
Вы можете переключаться между вкладками, наводя на них курсор мыши и вращая КП. Также можно перетаскивать вкладки с помощью клавиш Ctrl + ЛКМ и Alt + ЛКМ. В случае с клавишей Ctrl, расположение вкладки контакта будет запомнено и она всегда будет находится на этом месте. Нажатие СКМ на вкладке закрывает её.
Если щёлкнуть ЛКМ на значок клиента контакта в контакт-листе, то откроется окно с информацией о данном контакте.
Если нажать Ctrl + ПКМ на контакте в контакт-листе, то в меню появятся пункты авторизации, даже на авторизованных контактах.
Примечание: иногда помогает при проблемах с авторизацией.
Если щёлкнуть ЛКМ в свободной области контакт-листа и начать набирать текст (Ник или Имя контакта), то произойдёт автоматический поиск контакта из списка и выделение соответствующей строки.
Чтобы вернуться в нормальный режим просмотра, нажмите клавишу Esc.
Для быстрого удаления или перемещения в другую группу большого количества контактов, нужно вызвать диалог "Поиск по списку" комбинацией клавиш Alt + F3 (или Главное меню → Настройки → Поиск по списку), где возможно групповое выделение контактов при помощи ЛКМ и клавиш Ctrl и ⇧ Shift.
Примечание: требуется плагин Быстрый поиск.
Если вам нужно в список контактов добавить новый контакт, но в данный момент у вас отсутствует соединение с интернет, то это можно сделать и без него. Для этого открываем окно поиска и добавления контактов, вводим ID искомого контакта и нажимаем кнопку Искать с зажатой клавишей Ctrl. После чего появится этот контакт и вам останется только добавить его в список контактов.
Примечание: работает только с протоколом ICQ.
Если в качестве основного журнала сообщений используется "Журнал IEView", то размер шрифта в окне переписки можно изменить с помощью кнопок + и − справа вверху окна переписки.
Если в качестве основного журнала сообщений используется "Встроенный журнал TabSRMM", то размер шрифта в окне переписки можно изменить вращая КП с зажатой клавишей Ctrl.
Если в качестве основного журнала сообщений используется "История", то размер шрифта в окне переписки можно изменить только через Настройки → Тонкая настройка → Шрифты и цвета → История.
ЛКМ — Левая кнопка мыши СКМ — Средняя кнопка мыши
ПКМ — Правая кнопка мыши КП — Колесо прокрутки
