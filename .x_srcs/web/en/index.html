Installation recommendations

  • When choosing a folder for installation, please note that it is not recommended to use non-Latin letters anywhere in the path to the program.
  • It is recommended to run the build as an administrator if the build is installed in the Program Files system folder and User Account Control (UAC) is enabled in the OS, otherwise IEView templates might not work correctly.
  • It's strongly recommended that you enable font smoothing when using a handwriting font. Under Windows XP you can enable it in this way: click RMB on the desktop → Properties → Appearance → Effects – select ClearType from the Use the following method to smooth edges of screen fonts drop-down list.
  • If you installed Miranda into a new (clean) folder, you would need to create a new profile.
  • Anyway moving or copying of the old profile is strictly not recommended!

Update recommendations

  • If you have an older version of the assembly, and this installer includes the ability to upgrade this version, the assembly of course could be upgraded, but only on condition that the update will be installed by the installer. Otherwise, the assembly must be installed in a new folder and a new profile should be created.
  • Anyway moving or copying of the old profile is strictly not recommended!
