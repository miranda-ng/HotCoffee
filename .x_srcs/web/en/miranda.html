Miranda NG

Miranda NG (Next Generation) is a successor of a popular multi-protocol instant messaging client for Windows – Miranda IM.

Attention! Stable work of Miranda NG is guaranteed only within the same build, i.e. two sequential builds might be completely incompatible.
To install and update any Miranda NG components it's strongly recommended to use the PluginUpdater plugin.



Official site: https://www.miranda-ng.org
Official forum: https://forum.miranda-ng.org
Wiki: https://wiki.miranda-ng.org
