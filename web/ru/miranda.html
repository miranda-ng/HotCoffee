Miranda NG

Miranda NG (Next Generation) — преемник популярного мультипротокольного IM-клиента для Windows, Miranda IM.

Внимание! Стабильная работа Miranda NG гарантируется в пределах одного билда, т. е. все плагины одной компиляции совместимы с ядром, а два соседних билда могут быть полностью несовместимы.
Для установки и обновления любых компонентов Miranda NG рекомендуется использовать плагин PluginUpdater.
