Ответы на часто задаваемые вопросы

В.: Существует ли общий справочник или FAQ по Miranda NG?
О.: Общие вопросы и Глоссарий.

В.: Где находится Главное меню?
О.: По умолчанию кнопка "Главное меню" расположена в окне списка контактов вверху слева.
О.: Так же "Главное меню" можно вызвать кликнув ПКМ на значке статуса в трее.

В.: Как убрать анимацию в журнале IEView?
О.: Главное меню → Вид → IEView → Анимация. Скриншот

В.: Как сделать, чтобы Miranda запускалась только после ввода пароля?
О.: Главное меню → База данных → Установить пароль.

В.: Где хранится история контактов, которые не были добавлены в список?
О.: Главное меню → История → Системная история.

В.: Как выключить множество вкладок на панели задач в Windows 7?
О.: Настройки → Беседы → вкладка «Контейнеры» — снимаем галку с пункта «Использовать возможности панели задач Windows 7» → производим перезапуск. Скриншот

В.: Как заблокировать/разблокировать статус аккаунта?
О.: Меню в трее → Статус → ICQ → ICQ. Скриншот

В.: Как в окне диалога во время приёма/отправки сообщений отключить автоматическую прокрутку предыдущих сообщений?
О.: Клавиша F12 включает/выключает автоматическую прокрутку.

В.: Как добавить город в список контактов?
О.: Главное меню → Настройки → Добавить контакты… В поле Искать: выбираем Weather → в поле Город: вводим название нужного города (рекомендуется на латинице) → жмём кнопку Искать → выделяем найденный город → жмём кнопку Добавить в список. Скриншот

В.: Как добавить радиостанцию в список контактов?
О.: Главное меню → Настройки → Добавить контакты… Чтобы увидеть полный список радиостанций, предлагаемых в этой сборке, имя радиостанции рекомендуется не вводить. В поле Искать: выбираем mRadio → выбираем группу Настроить → жмём кнопку Искать → выделяем интересующую радиостанцию → жмём кнопку Добавить в список. Скриншот
 О.: Если есть желание добавить радиостанцию, которой нет в списке сборки, тогда открываем Настройки → Сеть → mRadio. В поле Ссылка на станцию (требуется) вводим ссылку радиостанции (также рекомендуется заполнить дополнительные поля с информацией о радиостанции) → жмём кнопку В список. Скриншот
